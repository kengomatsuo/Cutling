Makipag-ugnayan

Para sa mga tanong, feedback, o ulat ng bug, mag-email sa kenneth@matsuokengo.com.

Nilalayon kong tumugon sa loob ng 48 oras.

Pag-troubleshoot

Hindi lumalabas ang keyboard

Tiyaking naidagdag mo ang Cutling keyboard: pumunta sa Mga Setting > General > Keyboard > Keyboard > Magdagdag ng Bagong Keyboard > Cutling. Pagkatapos ay i-tap ang Cutling at paganahin ang “Allow Full Access.”

Hindi lumalabas ang mga cutling sa keyboard

Tiyaking naka-enable ang Buong Pag-access para sa Cutling keyboard. Kung wala ito, hindi mababasa ng keyboard ang iyong mga na-save na cutling mula sa lokal na storage.

Hindi gumagana ang pag-sync ng iCloud

Tingnan kung naka-enable ang iCloud sa iyong device (Mga Setting > [Your Name] > iCloud) at may iCloud access ang Cutling. Maaaring tumagal ng ilang sandali ang pag-sync upang magpalaganap ng mga pagbabago sa mga device.

Hindi naglo-load ang mga larawan sa keyboard

Ang mga cutling ng imahe ay nangangailangan ng Buong Access. Kung hindi pa rin lumalabas ang mga larawan, subukang alisin at muling idagdag ang Cutling keyboard sa Mga Setting.

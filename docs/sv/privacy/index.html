Sekretesspolicy

Gäller från: 8 mars 2026

Cutling samlar inte, lagrar eller överför någon av dina data till oss. Dina data stannar på din enhet eller synkroniseras privat via ditt personliga iCloud-konto (valfritt).

Översikt

Cutling är en tangentbordstillägg som låter dig spara och snabbt infoga textfragment och bilder som adresser, signaturer, QR-koder och annat ofta använt innehål. Denna policy förklarar hur appen hanterar dina data.

Datainsamling

Cutling samlar ingen personlig data alls. Ingen analitik, inga kraschrapporter, ingen användningsspårning, inga annonserings-ID:n. Appen innehåller inte några analytics eller tracking-ramverk från tredje part. Utvecklaren har ingen möjlighet att läsa, komma åt eller hämta dina data.

Dina fragment

Alla fragment (textutdrag och bilder) som du skapar lagras lokalt på din enhet. Dina data överförs aldrig till utvecklaren eller tredjepartsservrar.

iCloud-synkronisering (valfritt)

Cutling erbjuder valfri iCloud-synkronisering för att hålla dina fragment synkroniserade på dina Apple-enheter. När det är aktiverat:

  • Dina fragment synkroniseras via ditt personliga iCloud-konto med Apples privata CloudKit-databas
  • Endast du kan komma åt dina synkroniserade data via ditt Apple ID
  • Utvecklaren har ingen åtkomst till dina iCloud-data
  • Du kan inaktivera synkronisering när som helst i appens inställningar

När iCloud-synkronisering är inaktiverad förblir dina data strikt lokala på din enhet. iCloud-synkronisering tillhandahålls av Apple och är föremål för Apples villkor för iCloud.

Tangentbordstillägg

Cutling innehåller ett anpassat tangentbordstillägg som kräver fullständig åtkomst för att läsa från din enhets urklipp. Denna behörighet används endast för att låta dig klistra in och hantera dina sparade fragment. Tangentbordet loggar, registrerar eller överför inte dina tangenttryckningar eller urklippsinnehål. All urklippskomst sker lokalt på din enhet.

Tjänster från tredje part

Cutling använder inte några tjänster från tredje part, SDK eller API:er. Ingen data skickas till externa servrar.

Barnens sekretess

Cutling samlar ingen data från någon, inklusive barn under 13 år. Appen är säker för all åldrar.

Ändringar av dennaolicy

Om denna policy uppdateras kommer ändringarna att publiceras på denna sida med ett reviderat giltigt från-datum.

8 mars 2026: Uppdaterad för att återspegla tilläggningen av bildfragment och valfri iCloud-synkroniseringsfunktion. Klargjort att iCloud-synkronisering är valfritt och kan inaktiveras, och att synkroniserade data lagras på ditt personliga iCloud-konto.

Kontakt

Om du har frågor om denna policy kan du kontakta utvecklaren på kenneth@matsuokengo.com.

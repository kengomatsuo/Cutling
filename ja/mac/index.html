Mac用カトリング

ホットキー 1 つでクリップボードにアクセスできます。 カットリング - クリップボード キーボード はメニュー バーに常駐し、カーソルがある場所に表示されるため、フローを中断することなく、保存したスニペットを任意のアプリにドロップできます。

Mac 用ダウンロード

無料でダウンロードできます。 開発者 ID で署名され、Apple によって公証されています。 macOS 14 以降、Apple シリコンまたは Intel が必要です。

デスクトップ用に構築

メニューバーに常駐

カットリング - クリップボード キーボード は、ドックの外、邪魔にならないメニューバーにひっそりと鎮座しています。アイコンをクリックすると、保存したカトリングと最近のクリップボード履歴が表示されます。

グローバルホットキー

任意のアプリからショートカットを押すと、カーソルの位置にフローティング ピッカーが開きます。設定で任意のキーの組み合わせに再バインドします。

直接貼り付け

オプションで、カットリング - クリップボード キーボード が使用していたアプリにピックを直接貼り付けることができます。デフォルトではオフになっており、いつでも有効にするかどうかを選択できます。

クリップボード履歴

コピーしたものはすべて自動的にキャプチャされるため、何も失われません。保存しておきたいものはワンクリックで保存できます。

iCloud同期

iCloud をオンにして、保存したカットリングを iPhone、iPad、Mac 間で同期させます。オプションでプライベートです。

設計によりプライベート化

アカウントも分析も追跡もありません。クリップボードは Mac に保存されるか、同期を有効にしている場合は自分の iCloud に保存されます。

試してみる準備はできましたか?

Mac 用 カットリング - クリップボード キーボード をダウンロードして、再利用するすべてのものを 1 つのショートカットに保管します。

Mac 用ダウンロード

無料でダウンロードできます。 開発者 ID で署名され、Apple によって公証されています。 macOS 14 以降、Apple シリコンまたは Intel が必要です。

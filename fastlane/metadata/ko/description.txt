자주 쓰는 텍스트, 탭 한 번이면 끝. Cutling에 가벼운 키보드 확장을 추가하면, 저장해 둔 스니펫 — 주소, 계좌 정보, 이메일 서명, 신분증 번호, 반복해서 입력하는 모든 것 — 을 앱을 나가지 않고 바로 붙여넣을 수 있어요.

사용 방법
Cutling 열고, 스니펫 만들고, 키보드 켜면 끝. 주소를 입력하거나 계좌 정보를 보내야 할 때, Cutling 키보드로 전환해서 스니펫을 탭하면 돼요. 간단하죠.

왜 Cutling인가요?
계정 필요 없어요. 광고 없어요. 추적도 없어요. 같은 내용 반복 입력하는 번거로움을 없애주는, 빠르고 안전한 도구예요.
— 주소, 계좌 정보, 신분증, 자기소개 등 저장
— 키보드 확장으로 어떤 앱에서든 바로 붙여넣기
— iCloud로 모든 기기에 자동 동기화
— 만료일 설정으로 임시 스니펫 자동 삭제
— 컬러 라벨과 아이콘으로 스니펫 정리

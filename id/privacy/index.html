Kebijakan Privasi

Efektif: 8 Maret 2026

Cutling tidak mengumpulkan, menyimpan, atau mengirimkan data Anda apa pun kepada kami. Data Anda tetap ada di perangkat Anda atau disinkronkan secara pribadi melalui akun iCloud pribadi Anda (opsional).

Ringkasan

Cutling adalah ekstensi keyboard yang memungkinkan Anda menyimpan dan dengan cepat menyisipkan cuplikan teks dan gambar seperti alamat, tanda tangan, kode QR, dan konten lain yang sering digunakan. Kebijakan ini menjelaskan cara aplikasi menangani data Anda.

Pengumpulan Data

Cutling tidak mengumpulkan data pribadi apa pun. Tidak ada analitik, tidak ada laporan kerusakan, tidak ada pelacakan penggunaan, tidak ada pengidentifikasi iklan. Aplikasi ini tidak mengandung analisis pihak ketiga atau kerangka pelacakan apa pun. Pengembang tidak memiliki kemampuan untuk membaca, mengakses, atau mengambil data Anda.

Potongan daging Anda

Semua potongan (potongan teks dan gambar) yang Anda buat disimpan secara lokal di perangkat Anda. Data Anda tidak pernah dikirimkan ke pengembang atau server pihak ketiga mana pun.

Sinkronisasi iCloud (Opsional)

Cutling menawarkan sinkronisasi iCloud opsional untuk menjaga agar cutling Anda tetap tersinkronisasi di seluruh perangkat Apple Anda. Saat diaktifkan:

  • Pemotongan Anda disinkronkan melalui akun iCloud pribadi Anda menggunakan database CloudKit pribadi Apple
  • Hanya Anda yang dapat mengakses data yang disinkronkan melalui ID Apple Anda
  • Pengembang tidak memiliki akses ke data iCloud Anda
  • Anda dapat menonaktifkan sinkronisasi kapan saja di pengaturan aplikasi

Saat sinkronisasi iCloud dinonaktifkan, data Anda tetap bersifat lokal di perangkat Anda. Sinkronisasi iCloud disediakan oleh Apple dan tunduk pada Syarat dan Ketentuan iCloud Apple.

Ekstensi Papan Ketik

Cutling menyertakan ekstensi keyboard khusus yang memerlukan Akses Penuh untuk membaca dari clipboard perangkat Anda. Izin ini digunakan semata-mata untuk memungkinkan Anda menempelkan dan mengelola potongan yang Anda simpan. Keyboard tidak mencatat, merekam, atau mengirimkan penekanan tombol atau konten clipboard Anda. Semua akses clipboard terjadi secara lokal di perangkat Anda.

Layanan Pihak Ketiga

Cutling tidak menggunakan layanan, SDK, atau API pihak ketiga apa pun. Tidak ada data yang dikirim ke server eksternal.

Privasi Anak

Cutling tidak mengumpulkan data dari siapa pun, termasuk anak-anak di bawah usia 13 tahun. Aplikasi ini aman digunakan oleh segala usia.

Perubahan pada Kebijakan Ini

Jika kebijakan ini diperbarui, perubahan tersebut akan diposting di halaman ini dengan tanggal efektif revisi.

8 Maret 2026: Diperbarui untuk mencerminkan penambahan potongan gambar dan fitur sinkronisasi iCloud opsional. Memperjelas bahwa sinkronisasi iCloud bersifat opsional dan dapat dinonaktifkan, dan data yang disinkronkan disimpan di akun iCloud pribadi Anda.

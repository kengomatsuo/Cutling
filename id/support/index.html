Pemecahan masalah

Papan ketik tidak muncul

Pastikan Anda telah menambahkan keyboard Cutling: buka Pengaturan > Umum > Keyboard > Keyboard > Tambahkan Keyboard Baru > Cutling. Lalu ketuk Cutling dan aktifkan “Izinkan Akses Penuh.”

Potongan tidak muncul di keyboard

Pastikan Akses Penuh diaktifkan untuk keyboard Cutling. Tanpanya, keyboard tidak dapat membaca potongan yang Anda simpan dari penyimpanan lokal.

Sinkronisasi iCloud tidak berfungsi

Periksa apakah iCloud diaktifkan di perangkat Anda (Pengaturan > [Nama Anda] > iCloud) dan Cutling memiliki akses iCloud. Sinkronisasi mungkin memerlukan waktu beberapa saat untuk menyebarkan perubahan ke seluruh perangkat.

Gambar tidak dimuat di keyboard

Pemotongan gambar memerlukan Akses Penuh. Jika gambar masih tidak muncul, coba hapus dan tambahkan kembali keyboard Cutling di Pengaturan.
